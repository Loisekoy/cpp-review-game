# 程式闖關島

給國二學生使用的 Scratch 與 C++ 總複習網站。包含十個互動關卡：

1. Scratch 積木執行順序
2. Scratch 重複積木
3. C++ `cin` / `cout`
4. Scratch 條件判斷對照 C++ `if`
5. C++ 變數宣告與指定值
6. C++ `=` 與 `==`
7. C++ `%` 與奇偶數判斷
8. C++ `for` 迴圈追蹤
9. C++ `while` 無限迴圈除錯
10. C++ `while` 條件與更新綜合題

網站為純靜態 HTML、CSS、JavaScript，可直接部署至 GitHub Pages。

## 互動小遊戲

`mini-game/` 是「程式能量接力」小遊戲，學生每題選出正確的 Scratch 或 C++ 指令，答對後角色前進一格。題目涵蓋事件、重複、輸入輸出、變數、if、for、while，適合課堂總複習或分組搶答。
