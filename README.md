# 程式闖關島

給國二學生使用的 Scratch 與 C++ 總複習網站。包含十個互動關卡：

1. Scratch 積木執行順序
2. Scratch 重複積木
3. C++ `cin` / `cout`
4. Scratch 條件判斷對照 C++ `if`
5. C++ 變數宣告與指定值
6. C++ `=` 與 `==`
7. C++ `%` 與奇偶數判斷
8. C++ `for` 迴圈追蹤
9. C++ `while` 無限迴圈除錯
10. C++ `while` 條件與更新綜合題

網站為純靜態 HTML、CSS、JavaScript，可直接部署至 GitHub Pages。

## 程式貪食蛇小遊戲

`mini-game/` 是「程式貪食蛇」小遊戲，學生用方向鍵、WASD、手機按鈕或滑動控制蛇，吃到 `int`、`if`、`for`、`while`、`cin`、`cout` 等語法能量後會加分並變長。遊戲旁會顯示目前語法的簡短提示，適合課堂總複習穿插使用。
